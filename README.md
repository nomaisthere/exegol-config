# Exegol-Config

Just all the config files i use to work better with exegol.
If you don't already have Exegol installed, it's okay. You can change right now. Go give support to the community, and go [download](https://exegol.com/) !

If you want to know useful shortcuts and tips on exegol, go look at [this](https://docs.exegol.com/tips-and-tricks).

This is the beginning of my config files, goal is to implement foxyproxy default burp suite setup, lots of tools i find useful, and more. Ain't that deep, it's just cool.

## Config explained

This setup takes for granted you are in the `.exegol/my-resources/setup/` folder. Here is the list of tools / aliases used (all credits are below):

### Tools

- htb-utils
- hashid
- fidra

### Aliases

- expand-alias
- fakeclock
